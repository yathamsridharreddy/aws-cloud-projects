# CloudSmiths

## Description
Cloud-based platform to connect blood and organ donors with recipients in 
real time.

## Technologies Used
- AWS EC2
- AWS S3
- AWS RDS
- AWS Lambda
- Docker
- Python (Flask)
- MySQL
- Postman

## Features
- Full-stack cloud deployment
- Secure database integration
- Scalable architecture using AWS services

Academic / Internship project
